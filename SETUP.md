# Khod Bot — Build Guide

## Quick summary
- You build on your Mac (free) using GitHub Actions
- Mac builds run on `macos-latest` runners (free minutes)
- Windows builds run on `windows-latest` runners (free minutes)
- Releases are attached automatically to GitHub Releases

## What gets built
| Platform | Files |
|----------|-------|
| Mac Intel | `Khod Order Bot-x64.dmg`, `Khod Order Bot-x64-mac.zip` |
| Mac Apple Silicon | `Khod Order Bot-arm64.dmg`, `Khod Order Bot-arm64-mac.zip` |
| Windows | `Khod Order Bot Setup.exe` (installer), `Khod Order Bot Portable.exe` |
